### Run Locally

To run **jack-portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/jack-portfolio.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/jack-portfolio.git
```
